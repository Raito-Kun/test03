8. Các lệnh cơ bản của Claude:
/init  				: Tạo file CLAUDE.md để những rules & instruction
/model				: Chọn model sử dụng Default (Opus) /Sonet/Haiku
/ck:				: prefix của claudekit commands — tập hợp các lệnh shortcuts giúp làm việc nhanh hơn với Claude Code
	/ck:brainstorm	: Là lệnh để brainstorm giải pháp kỹ thuật trước khi code.
	/ck:build 		: hướng dẫn Claude build feature mới
	/ck:review 		: review code hiện tại
	/ck:test 		: viết test cho code
	/ck:plan 		: lập kế hoạch trước khi code
	/ck:docs 		: tạo documentation
	/ck:watzup 		: tóm tắt những gì đang xảy ra trong project
	/clear			: xóa context của conversation hiện tại, reset lại bộ nhớ trong session đó (dùng khi cần làm mới vấn đề, vd cần brainstorm nhiều lần cho kỹ)
