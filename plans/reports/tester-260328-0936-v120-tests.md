## Data Integrity Tests

### Lead Scoring Calculation
```
Base score: 30
+ Source bonus (referral=10, website=5, etc.)
+ Status bonus (qualified=20, won=25, lost=-10)
+ Phone bonus: +5 if present
+ Email bonus: +5 if present
+ Interaction bonus: min(callCount*2, 10)
+ Recency decay: -5 to -15 based on days since contact
= Final score capped [0, 100]
```
✅ All components verified

### Debt Tier Calculation
```
DPD mapping:
  0         → current
  1-30      → dpd_1_30
  31-60     → dpd_31_60
  61-90     → dpd_61_90
  90+       → dpd_90_plus
```
✅ All tiers tested

### Follow-up Lead Filtering
```
Criteria:
```
